A button represents an action a user can take. Buttons can be clicked or tapped to perform an action or to navigate to another page. Buttons in Spectrum have several variations for different uses and multiple levels of loudness for various attention-getting needs.

There are many button variants to choose from in Spectrum. The variant property controls the main variant of the button, and a few other boolean properties can be combined to apply sub-variants, e.g. quiet and logic.

Buttons can have a label, and icon, or both. An icon is provided by passing an icon component to the icon slot. A label can be provided as a child to the button.
